# 🧠 Tech Quiz App

A full-stack quiz application built with HTML, CSS, JavaScript, Node.js and Express.

## Features
- 10 Technology & Programming questions
- 15 minute global countdown timer
- Do Later system — skip and come back
- Instant correct/wrong highlighting
- Full results with score breakdown
- Professional light UI
- Fully responsive design

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Data: JSON

## How to Run
npm install
node server.js
Then open http://localhost:4000
